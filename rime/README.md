# rime

如果一个软件的配置文件是文本，并且经常使用，就会忍不住建一个仓库同步这些配置。

### squirrel

```bash
git clone --depth 1 https://github.com/Yee2/rime.git\
 ~/Library/Rime
```

### ibus-rime

```bash
git clone --depth 1 https://github.com/Yee2/rime.git\
 ~/.config/ibus/rime
```

### fcitx-rime

```bash
git clone --depth 1 https://github.com/Yee2/rime.git\
 ~/.config/fcitx/rime
```
